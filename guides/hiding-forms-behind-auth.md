# Hiding forms behind basic authentication

1. Go to the Form Builder Publisher.
2. Select the form you want to deploy.
3. Select the Config Params tab.
4. Using uppercase letters, enter USERNAME in the Name field.
5. Enter your chosen username in the Value field.
6. Select the green Add button to add the username.
7. Using uppercase letters, enter PASSWORD in the Name field.
8. Enter your chosen password in the Value field.
9. Select the green Add button to add the password.
10. Select the Deployments tab.
11. Select the green Deploy Now button in the Development environment.
12. On the next page you see, select Deploy now.
